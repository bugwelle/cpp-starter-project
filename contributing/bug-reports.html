Bug Reports¶

You found a bug? Time to report it!

This project’s issue tracker is available on GitHub: https://github.com/bugwelle/cpp-starter-project/issues

When opening a new issue, please send us:

  • the name of your operating system plus its version (e.g. Windows 8.1, Linux Mint 19, etc.)
  • steps to reproduce the bug

We’ll try to fix reported bugs as soon as possible!
